# Monopoly Taiwan

 <br>
**Best experience size: iPhone 11**

![banner](https://miro.medium.com/max/1920/1*YKB75IujbEesaSo_eveXdA.png)

## Features
* Photo wall gallery
* Trim, rotate, mirror
* Color adjustment
* Apply photo filter effect
* Add custom text field on photo
* Save/share/delete edited photo
